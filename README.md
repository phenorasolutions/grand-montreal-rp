# Grand Montréal RP V3

## Installation

1. Décompresse le ZIP.
2. Va dans le dépôt GitHub `phenorasolutions/grand-montreal-rp`.
3. Clique sur **Add file** puis **Upload files**.
4. Glisse tous les fichiers et dossiers de cette V3.
5. Confirme le remplacement des anciens fichiers.
6. Clique sur **Commit changes**.
7. Attends le nouveau déploiement Cloudflare.
